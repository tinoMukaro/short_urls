#  URL Shortener (TypeScript + Docker)

A minimal URL Shortener API built with **Express**, **TypeScript**, and **PostgreSQL**, containerized using **Docker**.  
This project was created as a small personal exercise to practice working with **TypeScript**, **Docker**, and database integration.

---

##  Tech Stack

- **TypeScript** ‚Äì for type-safe backend logic  
- **Express.js** ‚Äì lightweight Node.js web framework  
- **PostgreSQL** ‚Äì relational database for storing URLs  
- **Drizzle ORM** ‚Äì simple and type-safe ORM for SQL queries  
- **Docker & Docker Compose** ‚Äì for containerization and easy setup
